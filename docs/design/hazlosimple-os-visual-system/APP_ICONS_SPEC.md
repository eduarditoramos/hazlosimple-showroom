# HazloSimple OS — App Icons Spec

## Regla

Los iconos deben parecer apps instaladas, no botones con letras.

Estilo:
- cuadro redondeado glossy
- profundidad sutil
- microdetalle interno
- badge pequeño
- active ring/glow
- label debajo

No usar emojis grandes. Usar símbolos sobrios y microformas CSS.

---

## CRM

Label: `CRM Simple`  
Icon text: `C`  
Color: CRM blue  
Microdetalle:
- 4 mini barras horizontales o puntos ascendentes
- badge `24`
Estado:
- `3 seguimientos hoy`

Botón:
- `↗ Registrar seguimiento`

---

## Cocina

Label: `Cocina / POS`  
Icon text: `Co`  
Color: cocina red / deep tomato  
Microdetalle:
- 3 líneas tipo ticket/comanda
- badge `18`
Estado:
- `2 pedidos en cocina`

Botón:
- `✓ Marcar listo`

## Clínica

## Operación

Label: `Control Operativo`  
Icon text: `Op`
