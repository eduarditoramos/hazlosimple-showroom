# HazloSimple OS — Microdemo Specs

## Regla principal

Cada demo debe vender control visible, no solo interfaz.

Cada microdemo debe mostrar:

1. Métrica clave
2. Gráfica simple
3. Flujo del negocio
4. Registro vivo
5. Acción inmediata

---

# 1. CRM Simple

## Promesa comercial

No pierdas prospectos por falta de seguimiento.

## Inspiración

Salesforce simple, pero HazloSimple: claro, vendible, sin complejidad.

## Debe mostrar

### Métricas
- 24 leads
- 21% conversión
- $87k pipeline
- 3 seguimientos hoy

### Gráfica
Funnel / pipeline de conversión:

Leads → Contactados → Cotizados → Ganados

Valores sugeridos:
- Leads: 24
- Contactados: 18
- Cotizados: 7
- Ganados: 5

### Registro vivo
Mariana López  
CRM para equipo comercial  
$18,500  
Cotización enviada  
Llamar hoy 4:30 PM  
Probabilidad de cierre: 72%

### Dolor visible
3 seguimientos vencen hoy. Este lead se enfría si no se llama.

### Acción
`↗ Registrar seguimiento`

### Resultado simulado
Seguimiento registrado · próxima llamada agendada
